Mikrotik Dasar

Install Mikrotik RouterOS dan Windows XP

Create VM Mikrotik RouterOS
- New
- Name : Mikrotik RouterOS Type: Other Version: Other
- Klik Next
- Memory Size: 64MB, Klik Next
- Create Virtual Harddisk Now
- VHD(Virtual Harddisk), Klik Next
- Dinamic Allocated, 2GB, Create

Setting Mikrotik RouterOS
-Klik OS, Setting, Storage, Empty, Choose Virtual Optical Disk File (ISO File)
-System, First Bootorder Harddisk
-Klik Ok

Install
- Klik Start 
- Tekan a (Untuk memilih semua fitu)
- Tekan i (Untuk install)
- Tekan n (Untuk new  Configuration)

Selesai
